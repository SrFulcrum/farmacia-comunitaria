Site institucional da Farmácia Comunitária.

Arquivos:
- index.html
- Dockerfile

Como usar no EasyPanel:
1. Envie esta pasta para um repositório no GitHub.
2. No EasyPanel, no serviço App, conecte esse repositório.
3. Como o projeto já tem Dockerfile, o Easypanel vai buildar a imagem usando ele.
4. A porta interna do container é 80.
5. Depois configure o domínio no serviço e faça o deploy.
